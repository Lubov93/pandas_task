### Installation


1.Clone the repo
   ```sh
   git clone https://github.com/Lubov93/pandas_task.git
   ```
2. Create virtualenv and activate
   ```sh
   virtualenv venv
   source venv/bin/activate
   ```

3.Install packages
   ```sh
   pip install -r requirements.txt
   ```

4.Run file
   ```JS
   run main.py
   ```
